CKAN Data API

Truy cập dữ liệu nguồn thông qua API web với sự hỗ trợ truy vấn mạnh mẽ. Further information in the main CKAN Data API and DataStore documentation.

The Data API can be accessed via the following actions of the CKAN action API.

Tạo https://odp.cantho.gov.vn/api/3/action/datastore_create
Cập nhật / Chèn https://odp.cantho.gov.vn/api/3/action/datastore_upsert
Truy vấn https://odp.cantho.gov.vn/api/3/action/datastore_search

Ví dụ truy vấn (5 kết quả đầu tiên)

https://odp.cantho.gov.vn/api/3/action/datastore_search?resource_id=ff5951e4-ffd1-42bf-84bc-67cb315d5917&limit=5

Ví dụ truy vấn (kết quả có chứa 'jones')

https://odp.cantho.gov.vn/api/3/action/datastore_search?resource_id=ff5951e4-ffd1-42bf-84bc-67cb315d5917&q=jones

Một yêu cầu ajax (JSONP) đơn giản tới API dữ liệu bằng jQuery.

          var data = {
            resource_id: 'ff5951e4-ffd1-42bf-84bc-67cb315d5917', // the resource id
            limit: 5, // get 5 results
            q: 'jones' // query for 'jones'
          };
          $.ajax({
            url: 'https://odp.cantho.gov.vn/api/3/action/datastore_search',
            data: data,
            dataType: 'jsonp',
            success: function(data) {
              alert('Total results found: ' + data.result.total)
            }
          });
        import urllib.request
        url = 'https://odp.cantho.gov.vn/api/3/action/datastore_search?resource_id=ff5951e4-ffd1-42bf-84bc-67cb315d5917&limit=5&q=title:jones'  
        fileobj = urllib.request.urlopen(url)
